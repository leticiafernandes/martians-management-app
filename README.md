# Martians Management App

This app provide service for magament of Martians customers.

![Image of Marvin The Martian](./public/marvin-martian-icon.png)

<br>

### `Requirements:`
```
node >= 15.13.0
```

### `Running:`
```
$ npm install

$ npm start
```

### `Test:`
```
$ npm test
```

### `Production build:`
```
$ npm run build
```
